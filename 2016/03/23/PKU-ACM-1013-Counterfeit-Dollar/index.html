PKU ACM 1013 Counterfeit Dollar

发表于 2016-03-23   |  

题目大意

有标记着A到K的12个硬币，其中有一个假币。秤重三次，只能看出两边是否平衡。求哪个编号对应的硬币是假币，且求其轻重。

梳理

其实是个简化版的13球问题（相对其它12个球重量不同，3次求出，找称重的方法）。
由于这次称重的办法都由题目提供，所以只要很简单地输出判断结果就好啦。
给出一个新类型Coin。
Coin
char letter;//字母
enumType type;// true mayup maydown none
void setType(_type){
如果我是up，目标是down，那么就设成true，反之_type
}

坑

  1. 输入的获取。本来想按字符判断输入的，然而试了半天Java的System.in.read()，会把新行和回车区别对待造成识别与预期不同。最后用Scanner.next()+String.substring()拆一段字符= =
  2. 判断的错误。通过setType方法确实已经完成了大部分轻重判断，但忽略了一个点——两个硬币对比出现轻重不同时，其他硬币就全都是真。加入这步判断，人脑的尝试和计算机处理算完全对上了。
  3. 最后只要输出列表中第一个不是True的对象就可以。
