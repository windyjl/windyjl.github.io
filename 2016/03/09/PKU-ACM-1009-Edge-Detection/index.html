PKU ACM 1009 Edge Detection

发表于 2016-03-09   |  

这次题目都不想贴了，智商退化了之后有点干不过它。
大意是处理一张图像，每个像素点与周围八个比较，取最大的差值作为该位置的结果值。
考点其实是压缩
给的数据里有10^9这样的变态值。
然而提交后提示running time error。我就日了狗了，反复设几次return后又跳了内存超标的错误。
老子解出来很给面子了啊，别在我做回复训练的时候要求太高。Java还要限制10000KB
TODO 等老了之后或许有时间来优化，目测有这样几个点

1.检测用的二维数组设定固定长宽，不根据输入内容新建
2.点与周围差值比较时新建了8个int，这个应该也可以省
3.可以尝试直接从压缩后的结果开始计算，不考虑展开图像后的对比
